# 飛書模板市集

> 中文世界第一套專業的飛書 / 釘釘工作模板。

## 產品理念

飛書和釘釘是中文世界最重要的協作平台，但：
- ❌ 官方模板太少、太簡單
- ❌ 網路上的模板品質低落、沒維護
- ❌ 沒有針對中文企業需求的專業模板

**我們要做的：** 提供高品質、可客製化、持續更新的飛書/釘釘模板。
