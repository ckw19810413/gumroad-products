# 銷售儀表板模板

## 模板結構

### 📊 主要報表
| 報表名稱 | 用途 | 更新頻率 |
|---------|------|---------|
| 銷售總覽 | 本月銷售總額、目標達成率 | 每日 |
| 銷售漏斗 | 各階段客戶數、轉換率 | 每週 |
| 業績排行 | 銷售人員業績排行 | 每週 |
| 產品分析 | 各產品銷售額、利潤 | 每月 |

### 📋 客戶清單
| 欄位 | 說明 |
|------|------|
| 公司名稱 | 客戶公司 |
| 聯絡人 | 主要聯絡人 |
| 階段 | 潛在 → 接觸 → 提案 → 成交 |
| 預估金額 | 預計合約金額 |
| 預計成交日 | 預計簽約日期 |
| 下一步 | 下一步行動 |
| 負責人 | 負責銷售人員 |

### 📈 KPI 儀表板
- 月銷售額 vs 目標
- 客戶獲取數
- 平均合約金額
- 銷售週期天數
- 贏單率

## 使用教學

1. 點擊「使用此模板」複製到工作區
2. 將「客戶清單」填入你的實際客戶資料
3. 設定每個銷售人員的月目標
4. 每週更新一次漏斗數據
5. 每月初查看 KPI 儀表板

## 客製化指引

- 修改 KPI 公式以符合你的業務
- 增加額外的漏斗階段
- 調整視覺化顏色以符合品牌
