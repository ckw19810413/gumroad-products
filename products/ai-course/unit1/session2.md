## 2.1 什麼是 Prompt Engineering？

Prompt engineering 就是「如何跟 AI 溝通」的科學。

**錯誤的 prompt：**
```
寫一封信。
```

**好的 prompt：**
```
你是一位有 10 年經驗的商務顧問。請幫我撰寫一封專業、得體的繁體中文商務郵件。
情境：...
目的：...
對象：...
```

**差異在哪？** 好的 prompt 包含了：
1. **角色設定**（你希望 AI 扮演什麼角色）
2. **背景資訊**（情境、目的、對象）
3. **格式要求**（要什麼格式、什麼語言）
4. **約束條件**（字數限制、語氣要求）

---

## 2.2 五個核心 prompt 技巧

### 技巧 1：角色設定（Persona）

讓 AI 扮演特定角色，輸出會更專業。

```
❌ 寫一封行銷信
✅ 你是一位有 10 年經驗的 B2B 行銷專家，請寫一封行銷信
```

### 技巧 2：提供上下文（Context）

給 AI 更多背景資訊，輸出會更精準。

```
❌ 寫一篇文章
✅ 請幫我寫一篇關於 AI 在醫療領域應用的部落格文章，目標讀者是醫師和醫護人員
```

### 技巧 3：指定格式（Format）

要求特定格式，輸出會更好用。

```
❌ 給我一些建議
✅ 請用列表格式給出 5 個建議，每個建議包含：標題、說明、執行步驟
```

### 技巧 4：提供範例（Few-shot）

給 AI 一個範例，它會模仿風格。

```
❌ 寫社群貼文
✅ 請像下面這個風格寫一篇社群貼文：

範例：
「還在為 [問題] 煩惱嗎？💡 今天分享 3 個我親測有效的方法：
1. ...
2. ...
3. ...

[CTA]」

主題：[你的主題]
```

### 技巧 5：逐步思考（Chain of Thought）

讓 AI 一步一步思考，邏輯會更好。

```
❌ 這個問題怎麼解決？
✅ 請一步一步思考這個問題：
1. 先分析問題的本質
2. 列出可能的解決方法
3. 評估每個方法的優缺點
4. 給出最終建議
```

## 2.3 Prompt 模板

### 模板 1：通用寫作模板
```
你是一位[角色]。
請幫我[任務]。
背景：[背景資訊]
目標受眾：[誰會看]
格式要求：[格式]
語氣：[語氣]
字數：[字數限制]
```

### 模板 2：分析模板
```
請幫我分析[主題]。
分析面向：[面向1、面向2]
背景：[背景資訊]
要求：
1. [要求1]
2. [要求2]
3. [要求3]
```

### 模板 3：創意發想模板
```
請幫我發想[數量]個[類型]。
主題：[主題]
限制條件：[條件]
要求：
- 每個項目包含：名稱、說明、執行步驟
- 按[排序方式]排列
- [其他要求]
```

## 實戰練習

### 練習 1：改造你的 prompt

找一個你最近用 AI 寫的 prompt，用上面五個技巧改造它。

**改造前：**
```
[你原本的 prompt]
```

**改造後：**
```
[使用五個技巧改造後的 prompt]
```

### 練習 2：比較輸出

用同一個問題，用「差 prompt」和「好 prompt」各問一次，比較輸出差異。

**差 prompt：**
```
寫一篇關於 AI 的文章。
```

**好 prompt：**
```
你是一位科技記者。請幫我寫一篇關於 AI 在日常生活中的應用的文章。
目標受眾：一般民眾（非技術背景）
格式：部落格文章
字數：800-1000 字
要求：用淺白的語言、包含 3 個實際案例、結尾要有總結
```

比較兩個輸出的差異，記錄你學到了什麼。

## 本章重點

- Prompt engineering 是跟 AI 溝通的科學
- 五個核心技巧：角色設定、上下文、格式、範例、逐步思考
- 使用模板可以快速產出高質量 prompt
- 實際比較不同 prompt 的輸出是最好的學習方法
